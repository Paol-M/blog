YAGPDB by default supports tickets which can be opened using /ticket open. But reaction roles don’t work that easily. So I wrote some code to add this feature.

You can also watch the video (uses text to speech).

Guide

  1. In YAGPDB.xyz, select a server.
  2. Go to “custom commands” under core.
  3. Add a custom command that is triggered by reactions (added and removed)
  4. Paste the below code. Change from the lines 3 till 5 the 12345678 to the IDs of the actual channels and catagories in your Discord server. To get the ID of a channel or catagory, you should enable developer options in your Discord settings.

{{/* Admin Only Embed */}}
    {{$ao1 := cembed "description" (print $ao_Emoji " This ticket is now in Admin Only mode") "color" 1752220}}
    {{$ao2 := cembed "description" "This ticket is no longer in Admin Only mode" "color" 1752220}}

{{$IS_Admin:=false}}
